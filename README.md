# 狼人杀模拟器

用模特卡洛模拟狼人杀游戏，看看狼人全部假装平民，胜率究竟是多少。

## 角色

### 狼人

1. 每天晚上杀人顺序为：
    1. 预言家
    2. 女巫
    3. 其他神
    4. 预言家说过的好人（金水）
    5. 女巫说过的好人（银水）
    6. 非狼人警长
    7. 其他非狼人
    8. 绝对不杀队友
2. 每天白天就假装平民，和平民行动完全一致。
    1. 发言没有任何信息。
    2. 不假装自己是预言家。
    3. 不竞选警长。
    4. 投票的时候，跟着好人投，哪怕是队友。
    5. 绝对不自曝身份。

### 预言家

1. 每天随机查验一名未知玩家。（因为狼人全部假装平民，所以除了自曝身份以外的人，看上去应该都一样）
2. 100%竞选警长，且100%会被选为警长。（因为狼人全部假装平民，不会去假装预言家抢选票）
3. 每天白天都会说出自己查看的玩家身份，并且其他人100%相信它。（理由同第二点）
4. 每天都会说出今晚要查看的玩家，如果是好人，死了就把警长给它；如果是狼人，就给之前查看过的好人。如果之前没看到过好人，就给女巫。
5. 白天投票优先投给自己已经发现的狼人。如果没有，就随机投给未知玩家，并避开自己已经和将要查验的玩家。（理由同第一点）

### 女巫

1. 第一天晚上无条件救被杀的人，因为狼人绝不会杀队友。
2. 第一天白天竞选警长时会公开自己的身份，所有人100%相信它。因为狼人只会假装平民。
3. 第一天白天竞选警长时会说出自己救的玩家，所有人100%相信它。（理由同第二点）
4. 白天投票：
    1. 预言家活着：跟着预言家投票。
    2. 预言家死了：跟着警长投票。
    3. 如果自己是警长：就选一个自己觉得最坏的玩家投票。
5. 第二天晚上100%使用毒药，逻辑同白天投票。

### 猎人

1. 预言家死之前都不公开身份。预言家死后立刻公开。（或者从不公开）
2. 白天投票逻辑同女巫。
3. 被杀带走人选择一个自己觉得最坏的。

### 白痴

1. 从不公开身份
2. 白天投票逻辑同女巫。

### 平民

1. 从不公开身份
2. 白天投票逻辑同女巫。


## 策略分析

在该策略下，因该是偏向好人的。

### 好人优势

1. 狼人不会自曝也不会假装预言家和女巫，所以预言家和女巫的身份很容易被验证，从而获得警长和警徽流的优势。
2. 狼人不会自杀，所以女巫第一晚不会救错人。
3. 女巫第一晚死了也可以救自己。
4. 猎人被女巫毒死了也可以开枪。
5. 狼人投票都会跟着预言家或者警长投。

### 狼人优势

1. 白痴绝对不会自曝身份。
2. 假设狼人能够完美扮演平民。（其实这个应该不算很难）

## 最终结果

在100000次模拟中，如果猎人会公开身份，狼人胜率稳定在了55%。如果猎人从不公开，还会上升到66%。所以其实狼人杀这个游戏很简单，就假装平民就完了。

## 未来计划

让猎人和白痴有更多的策略。或者女巫不自曝身份，虽然我觉得这样子好人会更难。
